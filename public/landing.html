How It Works

Get started in minutes. No configuration required.

1

Install Locally

Clone the repository and install dependencies. Includes server, API, and dashboard UI.

2

Launch Server

Run npm start to launch both API and dashboard. Access at localhost:8787. Works offline.

3

Monitor & Optimize

View real-time metrics, receive alerts, and implement cost-saving recommendations.

Installation

Three commands to get started. 

# Clone the repository
git clone https://github.com/mosse692/LLM-Intel.git
cd LLM-Intel

# Install dependencies
npm install

# Start the server
npm start

# Open dashboard at http://localhost:8787
