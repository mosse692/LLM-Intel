Get Started in 3 Steps

Install, wrap your calls, and you're done.

# 1. Clone and install
git clone https://github.com/yourusername/llm-copilot.git
cd llm-copilot
npm install

# 2. Start the server
npm start

# 3. Open dashboard
open http://localhost:8787

Integrate with Your App

// Before: Direct OpenAI call
const response = await openai.chat.completions.create({
  model: "gpt-4",
  messages: [{ role: "user", content: "Hello" }]
});

// After: With LLM Copilot
const response = await fetch('http://localhost:8787/api/v1/infer', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    prompt: "Hello",
    model: "gpt-4.1",
    provider: "openai"
  })
});

Features

Everything you need to optimize your LLM spend.

🎯 Smart Recommendations

  • ✅ Detect expensive models on short outputs → suggest cheaper alternatives
  • ✅ Identify high token usage → recommend stricter limits
  • ✅ Catch reliability issues → enable fallback routing
  • ✅ Budget alerts → prevent cost overruns

🛡️ Production-Ready Resilience

  • ✅ Automatic retries with exponential backoff
  • ✅ Primary → Fallback model routing (gpt-4 → gpt-4-mini)
  • ✅ Request timeouts (10s default)
  • ✅ Multi-provider support (OpenAI, Groq, more coming)

📊 Real-Time Monitoring

  • ✅ Cost tracking (by model, endpoint, time)
  • ✅ Error rate & latency trends
  • ✅ Request volume & token usage
  • ✅ CSV import for historical data
